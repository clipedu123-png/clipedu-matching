### 컴포넌트

- `.hero-proof-strip` (신규) — Quick Proof Bar를 hero 하단에 통합

### 인터랙션
- 페이지 진입 시 기존 sequential fade-up 유지
- 핸드폰은 hero 진입 시 자동 롤링 시작
- LIVE 점 펄스 (기존 `livePulse` 재사용)

### 베리언트 탐색 포인트
> Hero는 4가지 큰 베리언트 축이 있음 — 조합으로 시안 4~6개 가능.

1. **핸드폰 위치**: 우측 (위 도해) / 좌측 / 우측 + 살짝 기울임(rotate -3°) / 우측 + 떠다니는 카드 데코
2. **핸드폰 크기**: 작게(220) / 중간(260) / 크게(300, hero가 짧아짐)
3. **Proof Strip 노출 방식**: hero 안 통합(위 도해) / hero 아래 별도 strip / hero 좌측 컬럼 안에 끼움
4. **그라데이션 배경**: 현재 radial glow / 우상단 mesh / 좌하단 mesh / 풀 화이트(미니멀)
5. **LIVE 인디케이터 위치**: 키커 위 / 핸드폰 옆 floating badge / hero 우상단 corner
6. **CTA 배치**: 좌측 컬럼 안 (위 도해) / hero 하단 풀폭 / 핸드폰 옆

### 제약
- h1 카피·핸드폰 v2 자체는 유지
- CTA 2개 (primary + secondary) 유지
- Pretendard 폰트
- 최소 `min-height: 100vh` (현재 유지)

### 기술 메모
- 핸드폰 자동 롤링은 v2의 `.phone-mock` / `.phone-track` / 70장 이미지 / `phoneScroll` 키프레임 그대로 재사용
- Hero용 축소 클래스 `.phone-mock--sm` 추가만 하면 됨

```
↓ (sticky 영역 끝나면 아코디언 부가 정보)
[▸ 유입경로]  [▸ 얼마나 진행되고 있나?]
```

### 컴포넌트
- `.how-sticky-wrap` (신규) — `display: grid; grid-template-columns: 380px 1fr; gap: 80px;`
- `.how-sticky-panel` (신규) — `position: sticky; top: 120px;` (NAV 높이 + α)
  - 좌측 패널 안: 섹션 타이틀 + 한 줄 설명 + 진행 인디케이터(점 4개) + 작동방식 자세히 보기 작은 CTA
- `.how-steps` (신규) — 우측 스크롤 영역
- `.how-step` (신규) — 한 단계 카드 (`.step` 패턴 발전형, 큰 사이즈)
  - 단계 번호 + 타이틀 + 설명 + 옵션 일러스트/아이콘
- 아코디언 부분: 기존 `.proj` 패턴 그대로

### 인터랙션
- 좌측 패널: sticky
- 우측 step 카드들: 스크롤 진입 시 페이드인 + 약간 translate
- 진행 인디케이터(좌측 점 4개): 현재 화면 정중앙에 가까운 step에 활성 표시 (IntersectionObserver)
- 모든 step 노출 후 sticky 해제 → 아코디언 부가 정보로 자연 전환

### 베리언트 탐색 포인트
> 이 섹션이 페이지 중반 임팩트 포인트. 베리언트 4~6개 가능.

1. **좌측 패널 콘텐츠 풍부도**: 미니멀(타이틀+점만) / 중간(설명 추가) / 풀(아이콘·작은 일러스트·CTA)
2. **진행 인디케이터 스타일**: 점 4개 / 진행 바(progress bar) / 숫자 카드 4개
3. **Step 카드 시각**: 텍스트만 / 좌측 컬러 액센트 바 / 일러스트 또는 스크린샷 동반 / 큰 번호 typography
4. **Step 간 구분**: hairline 구분선 / 큰 여백 / 연결선 (단계가 이어지는 시각)
5. **Step 카드 배경**: 흰 카드 / 배경과 동일(borderless) / 짙은 카드 (다크 mode 변주)
6. **일러스트 사용**: 없음 / 간단 아이콘 / 미니 시뮬레이션 UI / 핸드폰 미니 mockup (각 step별 화면)

### 제약
- 4 step 콘텐츠 순서·문구 유지
- 좌측 sticky · 우측 스크롤 구조 유지 (핫스팟의 핵심)
- 아코디언 부가 정보 2개는 sticky 영역 후에 (스크롤 흐름 방해 X)

### 기술 메모
- `position: sticky` + `top: 120px` (NAV 높이 64px + 여백)
- 좌측 패널 높이가 짧으면 sticky 영역 짧아짐 → 좌측 패널을 의도적으로 채워 우측 스크롤 길이와 맞춤
- `IntersectionObserver`로 우측 step 진입 감지 → 좌측 인디케이터 활성화

- CTA: `원생 매칭 신청하기` (큰 버튼)
- 보조: 또는 `먼저 카톡으로 상담받기` (작은 ghost — 옵션)

### 레이아웃 의도

**풀블리드(`100vw`) 다크 그라데이션 + 중앙 정렬 단순 구성**

```
══════════ full-bleed dark gradient ════════════
║                                              ║
║                                              ║
║         24시간 안에 연락드릴게요.              ║
║         (큰 typography, 그라데이션 텍스트)       ║
║                                              ║
║   지금 신청하면 다음 매칭 라운드에 바로 포함됩니다.  ║
║                                              ║
║              [원생 매칭 신청하기]              ║
║                (큰 solid 버튼)                ║
║                                              ║
║                                              ║
══════════════════════════════════════════════
```

### 컴포넌트
- `.pre-footer-cta` (신규) — `width: 100vw; padding: 160px 0;`
- 배경: `background: linear-gradient(165deg, #173A66, #102540, #15355C)` (contact 섹션과 같은 톤 또는 더 짙게)
- `.pre-footer-headline` — 큰 typography (clamp(2.5rem, 5vw, 4rem))
- `.pre-footer-cta-btn` — 큰 CTA 버튼

### 베리언트 탐색 포인트
> Case 1의 임팩트 마무리. 베리언트 4~5개 가능.

1. **배경**: 다크 블루 (위) / 풀 다크 #142844 / 그라데이션 메시 / 별빛(starfield)
2. **헤드라인 스타일**: 일반 흰색 / 그라데이션 텍스트 클립 / 큰 outline
3. **CTA 모양**: 일반 pill / 화살표 동반 / 외곽 그라데이션 보더 / 페이드 글로우
4. **장식 요소**: 핸드폰 미니 mockup 떠 있음 / 카톡 풍선 일러스트 / 미니 라이브 카운터 / 깔끔하게 비움
5. **헤드라인 크기**: huge (5rem) / medium (3rem)
6. **레이아웃**: 풀 중앙 정렬 / 좌 텍스트 + 우 데코

### 제약
- 풀블리드 (`width: 100vw`)
- 다크 톤 (페이지 리듬 변주)
- CTA 1개 메인 (선택적으로 ghost 보조 1개)
- 너무 화려하지 않게 (신뢰감 유지)

### 기술 메모
- 페이지 wrap 안에 두지 말고 `<section>`을 wrap 밖으로 빼서 풀블리드 처리
- 또는 wrap 안에 두고 `margin-left: calc(50% - 50vw); width: 100vw;` 트릭

## 14. Sticky CTA Bar

### 목적
페이지 길어도 신청 동선 항상 노출.

### 콘텐츠
- 좌측 텍스트: `지금 신청하면 24시간 내 연락드려요`
- 우측 CTA: 🏫 `원생 매칭 신청하기` → `#apply`

### 레이아웃 의도

**페이지 하단 fixed bar**

```
═════════════════════════════════════════════════
│ 지금 신청하면 24시간 내 연락드려요  [원생 매칭 신청] │
═════════════════════════════════════════════════
```

### 컴포넌트
- `.sticky-cta` (신규) — `position: fixed; bottom: 0; left: 0; right: 0;`
- 등장 트리거: HERO 지나서 등장, Apply 진입 시 사라짐 (JS)

### 베리언트 탐색 포인트
1. **위치**: 하단 풀폭 (위 도해) / 하단 우측 floating chip / 우측 사이드 vertical
2. **배경**: 다크 / 라이트 글래스 / 그라데이션
3. **콘텐츠**: 텍스트 + CTA (위 도해) / CTA만 / + 작은 라이브 카운터

### 제약
- 데스크탑 전용
- z-index가 NAV(50)보다 낮게 (40 권장)
- Apply 섹션 진입 시 자동 숨김
